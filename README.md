## Descripción
Proyecto de Programación Avanzada desarrollado utilizando Programación Orientada a Objetos (POO) y consumo de APIs con Python.

Se implementaron las clases:
- `Country`
- `CountryAPI`

La aplicación consume información de la API REST Countries y permite:
- Buscar países por nombre
- Consultar países por región
- Calcular densidad poblacional
- Comparar países entre sí
